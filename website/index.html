🚀 Meu primeiro site com Docker!

📦 O que é Docker?

Docker é uma plataforma que permite criar, empacotar e executar aplicações em ambientes isolados chamados containers. Esses containers contêm tudo o que a aplicação precisa para rodar: código, dependências, sistema operacional e configurações.

🛠️ O que é Docker Compose?

Docker Compose é uma ferramenta que permite definir e gerenciar múltiplos containers usando um único arquivo de configuração chamado docker-compose.yml. Com ele, você pode iniciar toda uma aplicação com apenas um comando: docker-compose up.

🌐 O que é o Apache (HTTPD)?

O Apache HTTP Server (ou apenas Apache) é um dos servidores web mais utilizados no mundo. Ele é responsável por entregar arquivos HTML, imagens e outros conteúdos via protocolo HTTP para os navegadores.

📁 Estrutura do Projeto

Este projeto possui 4 arquivos principais:

  • index.html: a página que você está vendo agora.
  • style.css: o arquivo que define a estilização da página.
  • script.js: o arquivo que dar a interatividade a página.
  • docker-compose.yml: define o container do Apache e aponta o diretório onde o HTML será servido.

🚀 Como executar este projeto?

Com Docker e Docker Compose instalados, basta abrir o terminal e executar os comandos:

docker-compose up -d

Depois, acesse http://localhost:8081 no navegador para visualizar este site rodando dentro de um container Apache!

✅ Objetivo do Desafio

O objetivo foi criar um container executando um servidor Apache que serve uma aplicação web HTML simples, utilizando Docker Compose.

Esse é um passo essencial para entender como funciona o empacotamento e deploy de aplicações em ambientes isolados.
